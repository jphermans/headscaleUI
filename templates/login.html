How to Create a Login Key

  1. Access your Headscale server terminal
  2. Run the command:
    headscale apikey create --expiration 8760h
    Replace 8760h with desired validity:
    • 24h = 1 day
    • 168h = 7 days
    • 720h = 30 days
    • 8760h = 1 year
    • 87600h = 10 years
  3. Copy the generated key and use it to log in
Security Note: Store your API key securely. It provides full access to your Headscale server.

If your key has expired, generate a new one using the instructions above.
